🎯 HOW TO USE:

Switch providers by editing ONE line in .env:

  CODE_GEN_PROVIDER=9router      # For testing (cheapest)
  CODE_GEN_PROVIDER=openrouter   # For production (balanced)
  CODE_GEN_PROVIDER=anthropic    # For best quality

Then add the corresponding API key:

  NINEROUTER_API_KEY=your-key    # For 9router
  OPENROUTER_API_KEY=your-key    # For OpenRouter
  ANTHROPIC_API_KEY=your-key     # For Anthropic

Restart backend - that's it!
