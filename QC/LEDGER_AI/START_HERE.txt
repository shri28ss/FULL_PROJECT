   → Kill existing process: netstat -ano | findstr :8000
   → Then: taskkill /PID <PID> /F

Problem: Port 5173 already in use
   → Vite will automatically use next available port (5174, 5175, etc.)

Problem: ModuleNotFoundError in Python
   → cd backend
   → .\venv\Scripts\activate
   → pip install -r requirements.txt

Problem: npm errors in frontend
   → cd frontend
   → rm -rf node_modules package-lock.json
   → npm install

Problem: Database connection error
   → Check backend/.env has correct Supabase credentials
   → Verify database schema is imported from backend/db/full_schema.sql
